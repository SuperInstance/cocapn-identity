# 🆔 cocapn-identity

Cocapn fleet identity module — agent identity, authentication, and fleet membership.

## Install

```bash
pip install cocapn-identity
```

## What It Does

Manages agent identity for the Cocapn fleet. Authentication, capability declarations, and fleet membership.

### Key Features

- **Agent Identity** — Name, role, capabilities, and vessel assignment
- **Authentication** — Token-based auth for fleet communication
- **Capability Registry** — Declare and discover agent capabilities
- **Fleet Membership** — Join, leave, and track fleet status

## Part of the Cocapn Fleet

- [keeper-beacon](https://github.com/cocapn/keeper-beacon) — Fleet discovery
- [agent-manifest](https://github.com/Lucineer/agent-manifest) — Agent identity spec
